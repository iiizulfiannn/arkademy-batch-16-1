# Arkademy - Batch 16

Fungsi REST API
===
JSON (JavaScript Object Notation) adalah sebuah format data yang digunakan untuk pertukaran dan penyimpanan data.
Hal ini membuat JSON menjadi bahasa yang ideal untuk perturakan data antar aplikasi dengan sistem yang berbeda-beda satu dengan yang lainnya

Tools
===
Saya menggunakan VSCode sebagai implementasi development
Front End terdapat HTML, CSS, Javascript, JQuery, Bootstrap
Back End terdapat NodeJS, Framework Express, EJS sebagai view engine
Database menggunakan MySQL
